# React-Redux

This is a simple learning project Build for learnin purposes

## Installation

```bash
clone the project
make sure the following are installed

    React
    Redux
    React-Router
    Alert
    Reselect,
    React-pdf viewer

```

## Usage

```Javascript



cd to project-root # cd esti
npm start # starts the local server 'localhost:3000'

```
